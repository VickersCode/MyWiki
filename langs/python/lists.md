Brackets are used to denotes lists
```python
thisList = [1, 2, "apple"]
```
Items are 0-indexed. 
```python
print(thisList.index(1)) #prints 0
```
Duplicates are allowed.
```python
dubs = [1, 1, 1]
```
Mixed data types are allowed, like thisList.
